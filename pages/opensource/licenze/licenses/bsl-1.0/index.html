Come applicare questa licenza

Crea un file di testo (solitamente chiamato LICENSE o LICENSE.txt) nella cartella principale del tuo progetto e copiaci il testo della licenza.

Nota: Boost raccomanda di aggiungere una nota in cima ad ogni file. La nota di riferimento può essere trovata nelle Boost Software License FAQ.
